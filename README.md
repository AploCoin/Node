# Node
## Installation
There is 2 methods of installation: (tested on Ubuntu 22.04) 
- Docker
- From source

### Docker
1. Update list of packages and install some need packages
'''
sudo apt update
sudo apt install -y nano mc git curl
'''
2. Install Docker from script

'''
docker install
'''

3. Clone this repo

'''
git clone... 
'''
